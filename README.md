# Aplikasi Prediksi Diabetes

Aplikasi web berbasis Streamlit untuk memprediksi penyakit diabetes menggunakan
model machine learning (Logistic Regression).

## Fitur
- Input data pasien
- Prediksi diabetes
- Deployment online menggunakan Streamlit Community Cloud

## Teknologi
- Python
- Scikit-learn
- Streamlit
- Google Colab

## Cara Menjalankan
Aplikasi dijalankan secara online melalui Streamlit Cloud.
